# CPNS Master AI

Platform persiapan CAT CPNS cerdas berbasis AI. Simulasi ujian realistis, analisis kelemahan detail, dan pembelajaran adaptif untuk membantu Anda lolos seleksi dengan skor maksimal.

## Features

- **AI Smart Tryout**: Soal menyesuaikan kemampuan user (Adaptive Learning).
- **Simulasi CAT Realistis**: UX dibuat semirip mungkin dengan CAT BKN dengan navigasi dan timer real-time.
- **Analisis Kelemahan Super Detail**: Breakdown per subtopik (Pancasila, UUD, Logika Numerik) dan insight AI.
- **Gamifikasi**: Sistem XP, Level, dan Streak untuk menjaga motivasi belajar.

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- Gemini AI (Google GenAI)
- Framer Motion
- Recharts
- Lucide React
